=============================================================================
                            React JS                    React Native
=============================================================================
CSS Styling           |  - Inline css                | - Inline & Object
                      |  - Style object  
                      |  - external
Apply style prop      | className                    | style
Create style object   |  Normal object               | using StyleSheet.create() method
Default css property  |  Free to use flexbox         | bydefault uses flexbox
Default flex property |  -                           | by default flexDirection is column
Create section        | <div /> Tag                  | <View /> component
Render text           | <p> <h1> Tag                 | <Text /> component
Input Tag             |  html input Tag              | <TextInput /> component
Input change event    | html onChange event           | onChangeText event
Button Tag            | html button Tag              | <Button /> component
Button click event    | html onClick event           | onPress event
Rendering list        | using html <ul><li> & map()  | using <Flatlist /> component
